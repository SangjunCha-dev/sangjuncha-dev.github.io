sqlcmd 사용법 (sqlcmd-guide)

Posted Sep 30, 2020 2020-09-30T14:41:18+09:00 by Sangjun Cha

sqlcmd

사용목적 : 대용량 sql 파일실행은 sqlcmd만 실행가능

sqlcmd 사용 예시

1

sqlcmd \
-S 127.0.0.1,1433 \
-i input_file[ , input_file2...] \
-o output_file \
-U login_id \
-P password \
-d db_name

-S : 접속할 IP,접속포트
-i : 실행할 sql 파일 지정
-i C:<filename>
-i \\<Share$>\ -i "C:\Some Folder\" -o : 실행결과 저장할 파일 지정 -U : 계정 -P : 패스워드 -d : Database 이름

windows 환경

  • 계정과 패스워드는 설정해주지 않으면 윈도우 기본계정으로 접속

linux 환경에서 대용량 파일 줄단위 분할

  • split -l 10000 “aaa.sql” “aaa.sql.”
